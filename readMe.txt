Follow the steps below to run the whole project:

1. Open SQL Workbench
2. Open "TransactionalData Generator" file and run the script.
3. Open "MasterData Generator" file and run the script.
4. Open "createDW" file and run the script.
5. Open Eclipse IDE and open the project zip folder named "warehouse_project".
6. Open the file "meshJoin.java" and run it.
7. The output will ask whether to use default values or not. (type n and enter the required links)
